# InsTracker — описание приложения

Структурированное описание мобильного приложения для трекинга статистики Instagram. Формат оптимизирован для чтения ИИ: заголовки, списки, короткие фразы.

---

## 1. Главная суть приложения

- **Трекинг статистики Instagram** — сбор и отображение метрик аккаунта (подписчики, подписки, лайки, охваты и т.д.).
- **Отслеживание отписавшихся** — выявление пользователей, которые отписались от профиля.
- **Аналитика профиля** — сводные данные и тренды для принятия решений по контенту и аудитории.
